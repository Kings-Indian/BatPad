PRODUCTION FILES FOR BATPAD
============================

This folder should contain the manufacturing files for your BatPad project.

REQUIRED FILES:
1. gerbers.zip - PCB manufacturing files (export from KiCAD)
2. Top.STEP or Top.STL - Top case part
3. Bottom.STEP or Bottom.STL - Bottom case part  
4. Middle.STEP or Middle.STL - Middle case part (if applicable)
5. firmware.uf2 - Compiled firmware file

INSTRUCTIONS:
- Export your PCB as Gerbers from KiCAD and compress into gerbers.zip
- Export each case part as individual STEP or STL files
- Compile your firmware to firmware.uf2 format
- Place all files in this folder
